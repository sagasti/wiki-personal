# Chrome silent Gemini Nano install

## Chrome Silent Gemini Nano Installation

- **What**: Google Chrome downloads a **4 GB Gemini Nano** model onto users' machines **without consent**
- **No opt-in**, no opt-out (short of enterprise tooling)
- **Auto-re-download**: If user deletes the model, Chrome re-downloads it automatically
- **Scale**: 1+ billion devices — environmental/energy cost is massive
- **Privacy researcher**: Exposed by "That Privacy Guy" — drew parallels to Anthropic Claude Desktop silent install case
- **HN reaction**: 1615 pts — strong community backlash
- **Source**: https://www.thatprivacyguy.com/blog/chrome-silent-nano-install/
- **Pattern**: Same playbook as other companies silently installing AI models; notable precedent at unprecedented scale
