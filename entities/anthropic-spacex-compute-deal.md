# Anthropic SpaceX compute deal

## Anthropic–SpaceX Compute Partnership

- **Date**: May 6, 2026
- **Deal**: Anthropic gets access to **all** compute capacity at SpaceX's **Colossus 1** data center
- **Scale**: 300MW power, ~220K NVIDIA GPUs
- **Orbital compute**: They are also exploring compute in space (orbital data centers)
- **Rate limit changes** (effective same day):
  - Claude Code 5-hour rate limits **doubled** for Pro, Max, Team, Enterprise
  - Peak hours limit reduction **removed** for Pro and Max
  - API rate limits **raised considerably** for Claude Opus models
- **Source**: https://www.anthropic.com/news/higher-limits-spacex
- **HN**: 218 pts
- **Context**: Part of Anthropic's strategy to own infrastructure at massive scale rather than depend on cloud providers
