# AlphaEvolve DeepMind Agent

## AlphaEvolve

DeepMind released **AlphaEvolve**, a coding agent powered by **Gemini** that scales impact across multiple scientific fields.

- Uses Gemini as the underlying LLM to evolve and improve code
- Achieves research-level advances in mathematics and science — not just code generation but genuine discovery
- 175 points on HN, significant community attention
- Represents a shift from "AI as coding assistant" to "AI as research collaborator"

### Significance
Comparable in ambition to AlphaFold: applying LLM-driven search to scientific discovery rather than just software engineering.
