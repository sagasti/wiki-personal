# FAL.ai

- **Qué es:** Cloud API para image generation (Flux, etc.)
- **Configurado en Hermes:** Sí, como backend para `image_gen` tool (mayo 2026)
- **API Key:** Guardada en `~/.hermes/.env` como `FAL_API_KEY`
- **Tool:** `image_gen` habilitado en Hermes
- **Modelos:** Flux schnell, Flux dev, etc.
