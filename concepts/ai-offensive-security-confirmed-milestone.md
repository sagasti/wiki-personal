# AI offensive security confirmed milestone

In May 2026, Google publicly confirmed that criminal hackers used AI to discover a major software vulnerability — the first high-profile confirmation that AI-powered offensive security is no longer theoretical.

This sits alongside defensive/offensive AI dynamics:
- **Offensive**: Criminal actors leveraging AI for vulnerability discovery (confirmed by Google)
- **Defensive**: Small models also finding vulnerabilities previously requiring Mythos-class tools ([[ai-disrupting-vulnerability-disclosure-cultures]])
- **Constructive**: Claude AI recovered an 11-year-old BTC wallet holding $400k USD, demonstrating AI capability in cryptographic recovery

The BitLocker YellowKey zero-day also emerged the same week via an anonymous leaker dropping Microsoft zero-days sequentially — unrelated to AI but part of the same deteriorating security landscape.
