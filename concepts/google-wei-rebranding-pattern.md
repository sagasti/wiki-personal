# Google WEI rebranding pattern

Google's **Web Environment Integrity (WEI)** proposal faced significant pushback from the web standards community in 2023-2024.

In May 2026, Google announced **Google Cloud Fraud Defense** — described as "the next evolution of reCAPTCHA" — which is essentially WEI repackaged with a QR code challenge flow where users scan a code to prove they're human.

This follows a clear pattern: Google retreats from controversial web standards proposals, then re-introduces the same functionality under a product/brand name rather than a standards track, making it harder for the community to object.

- Original WEI: browser-level attestation API (standards track, community veto)
- Fraud Defense: product-level QR code flow (bypasses standards process)

Source: [Private Captcha analysis](https://privatecaptcha.com/blog/google-cloud-fraud-defence-wei/)
